# Sample Images

Place sample images in this folder for testing the DupLens AI duplicate detection system.

## Recommended Test Cases

1. **Exact Duplicates**: Same image copied multiple times
2. **Resized Images**: Same image in different resolutions
3. **Cropped Images**: Images with slight cropping differences
4. **Color Adjusted**: Images with brightness/contrast changes
5. **Similar but Different**: Images of the same subject from different angles

## Supported Formats

- JPEG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)
- BMP (.bmp)
- WebP (.webp)
- TIFF (.tiff)

You can also upload a ZIP archive containing multiple images.
